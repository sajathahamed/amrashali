# Journalist Portfolio & News Site

A production-ready, front-end-only journalist portfolio and news site built with Next.js 14+, TypeScript, Tailwind CSS, and Framer Motion.

## Features

- 5 responsive pages: Home, Articles, Single Article, Projects, About/Contact
- Style Guide page showcasing the design system
- Modern editorial design with generous whitespace
- Dark mode support
- Smooth animations with Framer Motion
- Full accessibility support
- Reading time estimator
- Share buttons
- Search and filtering

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Tech Stack

- Next.js 14+ (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Google Fonts (Playfair Display, Inter)

## Project Structure

```
├── app/              # App Router pages
├── components/       # Reusable components
├── data/            # JSON data files
├── lib/             # Utility functions
└── public/          # Static assets
```
